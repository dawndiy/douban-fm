# Douban FM

Douban FM for Ubuntu Touch. 

## About

An unofficial client of Douban FM. You can select the channels you like to play songs and share it to Sina Weibo.

- Playing songs from Douban FM
- Sync offline songs.
- Share to Sina Weibo.
