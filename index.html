安装

1. 使用安装包

下载: 安装包

用adb把安装包传到手机home目录里面: adb push <PACKAGE-FILE> /home/phablet/
运行 adb push pkcon install-local --allow-untrusted <PACKAGE-FILE>进行安装。

2. Ubuntu Store

在手机中搜索豆瓣FM或douban-fm，包名为 douban-fm.ubuntu-dawndiy 的应用进行安装。

Installation

1. Use click package

Download: Click

copy the .click file to phone's home folder via adb: adb push <PACKAGE-FILE> /home/phablet/
run adb push pkcon install-local --allow-untrusted <PACKAGE-FILE> to install.

2. Ubuntu Store

Search 豆瓣FM or douban-fm in Ubuntu Store on Ubuntu Phone, package name is douban-fm.ubuntu-dawndiy.
